# Technology Stack

## Core Technologies

- Language: Rust 2021 edition (AIRS ecosystem alignment)
- CLI Framework: clap v4 with derive features for clean command definitions
- Async Runtime: tokio for efficient file I/O operations
- Serialization: serde with YAML support for metadata parsing
- Error Handling: anyhow for ergonomic error management

## Output & Formatting

- Terminal Styling: colored for rich terminal output
- Unicode Support: unicode-width for proper text width calculation
- Terminal Detection: terminal_size for responsive formatting
- Text Processing: regex for content parsing and extraction

## Quality & Testing

- Testing Framework: Rust built-in testing with integration tests
- Documentation: cargo doc with comprehensive examples
- Linting: clippy for code quality
- Formatting: rustfmt for consistent code style
